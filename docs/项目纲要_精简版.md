# 二次元桌面助手 - 精简纲要

> 基于 Java + Spring Boot 的兴趣驱动学习项目

---

## 项目目标

打造一个能"看懂"你在做什么的二次元桌面助手：
- 监控电脑状态（CPU/内存/当前窗口）
- 语音播报与提醒（自定义声线）
- 即时通讯机器人交互（QQ/钉钉）
- Live2D 虚拟形象展示

---

## 技术架构

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  Python Agent │     │ Electron App │     │  QQ 机器人   │
│  (数据采集)   │     │ (Live2D形象) │     │ (远程交互)   │
└──────┬───────┘     └──────┬───────┘     └──────┬───────┘
       │                    │                    │
       └────────────────────┼────────────────────┘
                            │
                     ┌──────▼───────┐
                     │ Spring Boot  │
                     │   后端服务    │
                     └──────────────┘
```

## 开发阶段

| 阶段 | 目标 | 技术栈 | 预计时间 |
|------|------|--------|----------|
| **Phase 1** | MVP验证 | Spring Boot + OSHI | 2-3h |
| **Phase 2** | 窗口监控 | Python + pywin32 | 3-4h |
| **Phase 3** | 语音合成 | GPT-SoVITS | 3-4h |
| **Phase 4** | 智能响应 | 状态机 | 4-5h |
| **Phase 5** | 虚拟形象 | Electron + Live2D | 4-6h |
| **Phase 6** | QQ集成 | NapCatQQ | 5-6h |
| **Phase 7** | Web界面 | Vue/React | 可选 |

---

## 核心接口

```
GET  /status              # 获取系统状态（CPU/内存）
POST /report/window       # 上报活动窗口
POST /tts/speak           # 语音合成
GET  /activity/current    # 查询当前活动
WS   /ws/avatar           # Live2D WebSocket
```

## 环境要求

- JDK 17+
- Maven 3.8+
- Node.js 18+
- Python 3.10+
- Docker（GPT-SoVITS 部署）

---

## 快速开始

```bash
# 1. 启动后端
cd backend && mvn spring-boot:run

# 2. 启动 Agent
cd agent && python window_monitor.py

# 3. 启动桌面应用
cd desktop-app && npm start

# 4. 验证
访问 http://localhost:8080/status
```
