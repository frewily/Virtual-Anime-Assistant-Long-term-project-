# Live2D 模型目录

此目录用于存放 Live2D 模型文件。

## 模型结构示例

```
hiyori/
├── hiyori.model3.json    # 模型配置文件
├── hiyori.moc3           # 模型数据
├── textures/             # 纹理文件
│   └── texture_00.png
└── motions/              # 动作文件
    ├── idle_01.motion3.json
    └── tap_body.motion3.json
```

## 获取模型

1. Live2D 官方示例模型：https://www.live2d.com/en/download/sample-data/
2. Booth 平台：https://booth.pm/ja/search/live2d

## 注意事项

- 模型文件较大，建议使用 Git LFS 管理
- 注意模型版权，仅使用授权模型
